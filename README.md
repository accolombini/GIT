#### `.gitignore`

- Use para manter arquivos desnecessários fora do repositório.
- Arquivos comuns em `.gitingore`:
  - Arquivos temporários geradps por IDEs (Como PyCharm)
  - Logs, dados de cahce, arquivos de build.
